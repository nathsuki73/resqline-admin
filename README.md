# ResqLine Admin Frontend

The ResqLine Admin Frontend is a web-based dashboard built with Next.js. It serves as the administrative interface for managing and monitoring data from the ResqLine emergency reporting system.

---

## Overview

This project provides an interface for administrators to view, manage, and monitor emergency reports, user accounts, and other system data. It connects to the ResqLine backend API for all data operations.

---

## Tech Stack

- **Next.js** – Framework for server-side rendering and routing
- **React** – UI library for building components
- **TypeScript** – Type safety
